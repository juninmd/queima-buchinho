# Local Stickers Fallback

Place `.webp` or `.gif` sticker files here.

The bot uses these as fallback when Giphy API is unavailable or rate-limited.

## Naming Convention

Files are automatically mapped to categories:
- `celebration.gif`, `celebration.webp` - for workout celebration
- `fire.gif`, `fire.webp` - for general motivation
- `meme.gif`, `meme.webp` - for humor/memes
- `roast.gif`, `roast.webp` - for sarcastic responses
- `cardio.gif`, `cardio.webp` - for cardio activities
- `water.gif`, `water.webp` - for hydration reminders
- `trophy.gif`, `trophy.webp` - for achievements

## Recommended

- Format: WebP (best compression) or GIF
- Size: Under 500KB
- Telegram sticker size: 512x512px
