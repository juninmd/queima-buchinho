# Assets Directory

This directory contains motivational content used by the bot:

- `motivation.mp3` - Motivational audio file sent when user hasn't trained
- `motivation.jpg` - Motivational image sent when user hasn't trained

## How to add your own files:

1. Add an MP3 audio file named `motivation.mp3` with a motivational message
2. Add a JPG image named `motivation.jpg` with a motivational image

The bot will automatically use these files when sending motivation messages.
