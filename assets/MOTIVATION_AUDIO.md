# Placeholder for motivational audio

To use the motivational audio feature, add a file named `motivation.mp3` in this directory.

The audio should be a motivational message that will be sent to users who haven't trained yet.

Recommended:
- Format: MP3
- Duration: 10-30 seconds
- Size: Under 50MB (Telegram limit for voice messages)
- Content: A motivating voice message encouraging the person to exercise

Example message:
"Vamos lá! Não desista agora! Você é mais forte do que pensa. Levante-se e treine!"
