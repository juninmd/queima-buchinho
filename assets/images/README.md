# Local Images Fallback

Place `.jpg`, `.png`, or `.gif` image files here.

The bot uses these as fallback when Giphy API is unavailable or rate-limited.

## Naming Convention

Files are automatically mapped to categories:
- `motivation.jpg` - general motivation
- `workout_motivation.jpg` - fitness related
- `celebration.jpg` - celebrations
- `meme.jpg` - memes
- `roast_meme.jpg` - sarcastic content
- `hydrate.jpg` - hydration
- `morning_motivation.jpg` - morning reminders

## Recommended

- Format: JPG or PNG
- Size: Under 5MB (Telegram limit is 10MB)
- Dimensions: 800x600px or similar aspect ratio
